ユーザー様からのご要望に基づき、LP（`iitomos-tools/index.html`）のテキストと実際のツール機能の整合性監査、および販売・価格戦略の提案を実施します。

## 1. ツール機能とLPテキストの監査結果
`banbi/qr/popup.js` と `banbi/translator/script.js` の実際のソースコードを監査しました。
* ✅ **QRツール**: Bitlyやv.gdを用いた短縮URL発行機能がコードに実装されており、「短縮URLもワンクリックで同時発行可能」というLPの記述は **100%正確** です。
* ✅ **翻訳＆プロンプトメーカー**: 自動言語判定、Update Original（プロンプト洗練）機能、ワンクリックコピー機能が実装されており、LPの記述と **完全に一致** しています。

**結論**: ツール側で「LPで謳っている機能」はすべて完璧にカバーされており、誇大広告等の心配は一切ありません。追加修正は不要です。

## 2. 価格表示の更新プラン（HTMLの改修）
今回モニター価格として「1,000円（2ツールセット）」での販売をご検討されているとのこと、非常に良い戦略です。
コンバージョン（購入率）を最大化するためには、「定価（アンカリング）」を併記し、現在がどれだけお得かを視覚的にアピールすることが重要です。

### [MODIFY] `iitomos-tools/index.html`
LPの価格表示セクション（`.pricing-card`）を以下のように更新します。

#### 変更内容
1. **プラン名**: `Starter License — 買い切り` → `Starter License — 買い切り（先着モニター価格）`
2. **定価の併記（打消し線）**: `通常定価 ¥3,980` などの定価をグレーの打消し線で表示し、その下に大きく `¥1,000` を配置します。
3. **注意書き**: `※ 料金はLINEにてご案内します` → `※ 買い切り（ランニングコストなし）`
4. **収録内容**: 現在のリスト（QR＆翻訳を含み、他は追加購入可能とする記述）は今回の「2ツールセットで1,000円」という内容に完璧に合致しているため、そのまま使用します。

## 3. 販売戦略・マーケティングのアドバイス
モニター価格（1,000円）でリリースされる際の販売戦略として、以下の手法をおすすめします。

1. **アンカリング効果の活用**: 
   「通常価格3,980円のところ、最初の50名様（または今月末まで）限定で、モニター価格1,000円で提供します」と謳うことで、お得感や限定感を強調でき、即決を促せます。
2. **「お声」の収集（モニターの本来の目的）**:
   モニター価格で販売する代わりに、「使ってみた感想をLINEで一言教えてください」とお願いする仕組みにすると自然です。集まった声は、後日定価で販売する際のLPの「お客様の声」として強力な武器になります。
3. **ツールの『追加拡張性』をフックにする**:
   「今回は初期の2ツールのみですが、今後『画像分割』や『スニペット管理』が追加された際は、Starter Licenseをお持ちの方は優待価格（または無料）で追加できる」といった特典をつけると、よりライセンスの価値が跳ね上がります。
